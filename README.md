## Project Description
This project is a **Personal Task Manager mobile application** developed using the **Flutter framework**.  
The application enables users to create, view, complete, and delete personal tasks in a simple and user-friendly manner.

In line with the course requirements, the application was developed **without any backend service, database, or local storage**. All task data exists **only during application runtime**.

---

## Aim of the Project
The aim of this project is to demonstrate practical understanding of:
- Mobile application development using Flutter
- User interface design with core Flutter widgets
- Screen navigation using the Navigator class
- State management using `StatefulWidget` and `setState()`

---

## Application Features
- Splash screen displayed on application launch
- Home screen showing a list of tasks
- Add task functionality with input validation
- Task details view
- Mark task as completed
- Delete task with confirmation dialog
- Snackbar notifications for user actions
